##  1.제작 기간

2022.03.18 ~ 2022.03.19


## 2. 사용 기술

* Framework: React


## 3. 핵심 기능

* 요일 정렬
  - <br>현재 요일을 기준으로 순서를 정렬합니다.
  
* 요일별 별점 랜덤
  - <br>메인 페이지의 별점은 랜덤으로 배치합니다.

* 별점남기기
  - <br>해당 요일의 별점을 클릭을 통해 정할 수 있습니다.
